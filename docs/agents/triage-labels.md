# Triage labels

Five canonical triage roles, mapped to GitHub label names:

| Role | Label | Color |
|------|-------|-------|
| Needs triage | `needs-triage` | `#5319e7` |
| Needs info | `needs-info` | `#fef2c0` |
| Ready for agent | `ready-for-agent` | `#0e8a16` |
| Ready for human | `ready-for-human` | `#bfdadc` |
| Won't fix | `wontfix` | `#ffffff` |
